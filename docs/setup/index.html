Add SmokeBot

Simple steps to get SmokeBot working in your server.

1) Invite the bot

Click Add SmokeBot, select your server, and approve the requested permissions.

2) Grant permissions

Ensure SmokeBot can read/send messages, manage threads, and manage roles where you need it.

3) Configure tickets

Use /ticket to post the menu, then /addticketcategory to add categories.

4) Seed replies

Add snippets (/addsnippet) or auto replies (/autoreply set) for common responses.

5) Run /help

See all commands and teach staff the key ones for tickets, snippets, and moderation.
