Privacy Policy

Last updated: 12/8/2025

SmokeBot is a Discord moderation and support assistant developed for deployment by individual server owners. This Privacy Policy explains what limited information the bot processes when it operates in a Discord server, how that information is used, and the choices available to server administrators and members.

1. Information We Process

SmokeBot only accesses data that Discord makes available to bots in servers where it has been invited. The bot does not collect information outside of Discord. Depending on the features enabled by server administrators, SmokeBot may process:

1.1 Ticketing system

  • Discord user IDs for the member who created the ticket and staff who interact with it
  • Category selection, ticket status, creation and closure timestamps
  • The channel or thread ID that hosts the ticket conversation
  • Any messages voluntarily posted in the ticket thread (handled directly on Discord - not stored by the bot)

1.2 Reaction roles

  • Message IDs, emoji identifiers, and the role ID that should be granted or removed
  • Guild (server) IDs necessary to apply the role

1.3 Snippets

  • Per-guild snippet triggers and the text that should be posted when a trigger is used
  • Whether a snippet is dynamic and any placeholder structure configured by server staff

1.4 Persistent pins

  • The text content configured for the persistent pin
  • The IDs for the message, channel, guild, and the administrator who set the pin

1.5 Moderation utilities

  • IDs and parameters needed to complete actions like timeout, kick, ban, or purge; these are not stored after completion.

SmokeBot requires access to message content in servers where administrators enable snippet functionality or pin reposting. This access is limited to channels where the bot has permission so it can detect snippet triggers and keep tracked pins up to date.

2. How We Use Information

  • Provide and manage ticketing, snippet, pin, reaction role, and moderation features requested by server administrators.
  • Maintain configuration across restarts by storing the minimum required settings in local JSON files on the host machine.
  • Respond to slash commands and interaction events routed through Discord.

SmokeBot does not use the data for analytics, advertising, or any secondary purposes.

3. Where Information Is Stored

All configuration data is stored on a Google Cloud server operated for SmokeBot. No data is transmitted to external third parties other than Discord's platform APIs. The operator secures the cloud environment and access to the stored data.

4. Data Retention and Deletion

Stored configuration data remains until a server administrator edits or removes it or deletes the relevant files. Ticket threads remain on Discord until staff archive or delete them.

5. Legal Basis

For EU/EEA servers, SmokeBot processes the limited personal data described above under the legitimate interest of the administrators who deploy the bot.

6. Security

The bot relies on the hosting operator for physical and network security. Administrators should safeguard the Discord bot token and host access.

7. Children's Privacy

SmokeBot does not knowingly collect personal information from children. Discord requires users to be at least 13 years old (or the minimum age in their jurisdiction).

8. Third-Party Services

SmokeBot interacts exclusively with Discord's APIs. Use of the bot is also subject to the Discord Terms of Service and Privacy Policy.

9. Changes to This Policy

We may update this Privacy Policy. When changes are made, the "Last updated" date will be revised. Material changes will be communicated to server administrators.

10. Contact

SmokeBot is hosted on Google Cloud; the primary contact is the individual or team operating the SmokeBot instance for your server.
